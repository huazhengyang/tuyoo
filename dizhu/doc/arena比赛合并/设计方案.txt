1. Arena比赛支持多种配置(报名费，奖励)
2. 客户端报名、获取比赛信息时需要携带透传参数（用于标识使用哪种配置）
	{"cmd":"m_signs", "params":{"signinParams":{"mixId":"0"}}}}
3. 房间列表配置
	每种比赛配置一个房间列表项，加入透传参数配置（老版本只能配置一个）
4. 用户报名后使用的配置就固定了
5. 中途退赛
	玩家中途退赛后该比赛的所有配置的房间列表都无法进入
6. 老版本支持
	老版本只支持一种配置

配置
	{
		...
		"feeRewardList":[
			{
				"fees":[],
				"mixId":"0",
				"rank.rewards":[],
				"roomName":"100元",
			},
			{
				"fees":[],
				"mixId":"1",
				"rank.rewards":[],
				"roomName":"500元",
			}
		]
	}
